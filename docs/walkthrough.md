# restapi-app 対話型コンソールアプリへの改修完了

ユーザーから提供されたご要望に基づき、モデルロード用 API を統合した対話型コンソールアプリケーションへと `restapi-app` を改修しました。

## 主な変更点

*   **1. 表示と選択機能**: `main.py` 実行時に利用可能なモデルの一覧が出力され、ユーザーが標準入力 (`input`) で番号を入力してモデルを決定できるようになりました。
*   **2. モデルロード機能**: `load_model` 関数を新設し、`POST /api/v1/models/load` に対して指定されたモデルを LM Studio にロードさせる処理を追加しました。
*   **3. チャットのプロンプト入力**: 固定メッセージではなく、ロード後にその場でチャットメッセージを入力できるようにフローを改善しました。
*   **4. テストコードへの反映**: `test_main.py` にて `load_model` 関数のテストを追加しました。さらに、`main()` 関数の結合テストでは、標準入力をモック化して自動テストがこれまで通り成立するように調整済みです。

## 実行方法

> [!TIP]
> 対話型アプリのため、これまでと同様の操作で実行いただけます。
> **アプリ実行:** `bash start_app.sh`
> **自動テスト:** `bash start_ut.sh`
